# RealityHD

Place objects in and around your home! Amazing AR practice project!

### Technologies Used

- Swift
- RealityKit / ARKit
- SwiftUI / UIKit / Combine
- Object Occlusion / People Occlusion
- Model Conversion
- Object Interactivity
- Scene Persistence
